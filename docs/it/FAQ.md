# Domande Frequenti (FAQ)

## Generale

### Qual è lo scopo di questo sito web?
Il sito serve come portfolio professionale per mostrare competenze, esperienze e progetti nel campo dell'ingegneria delle infrastrutture, virtualizzazione, sviluppo software e analisi dei dati.

### Come posso visualizzare il sito?
Puoi accedere al sito online a **[umeglio.github.io](https://umeglio.github.io)** oppure clonare il repository per visualizzarlo localmente.

## Installazione e Configurazione

### Quali sono i prerequisiti?
- Git (per clonare il repository)
- Un editor di testo (opzionale, per modifiche)
- Un browser moderno

### Come configuro il sito localmente?
Segui la [Guida di Installazione](INSTALLAZIONE.md) per i dettagli step-by-step.

### Posso usare il sito offline?
Sì, puoi scaricare il repository e visualizzare i file HTML localmente nel tuo browser.

## Modifica e Personalizzazione

### Come modifico il mio profilo?
Modifica i file `index.html` (italiano) e `index_en.html` (inglese). Consulta la [Guida di Utilizzo](UTILIZZO.md) per i dettagli specifici su ogni sezione.

### Come aggiungo una nuova competenza?
Modifica la sezione "skills" nei file HTML. Aggiungi un nuovo elemento con il nome e la percentuale della competenza.

### Posso modificare i colori e lo stile?
Sì, il CSS è incluso nei file HTML. Puoi modificare le proprietà nel tag `<style>` per personalizzare l'aspetto.

### Come cambio l'immagine del profilo?
Sostituisci il file `avatar.png` con la tua immagine, mantenendo lo stesso nome di file e assicurandoti che si trovi nella directory principale.

## Lingue

### Il sito supporta altre lingue oltre a inglese e italiano?
Attualmente supporta solo italiano (index.html) e inglese (index_en.html). Per aggiungere altre lingue, consulta la [Guida per i Contribuenti](CONTRIBUIRE.md).

### Come faccio a mantenere sincronizzate le versioni in diverse lingue?
Quando fai modifiche, aggiorna sia `index.html` che `index_en.html` con il contenuto corrispondente. Mantieni la stessa struttura HTML per facilitare il mantenimento.

## Pubblicazione e Hosting

### Dove è ospitato il sito?
Il sito è ospitato su **GitHub Pages** al dominio `umeglio.github.io`.

### Quanto tempo impiega il sito a aggiornarsi dopo il push?
Generalmente entro 1-2 minuti. Se non vedi le modifiche, prova a svuotare la cache del browser (Ctrl+Shift+Canc).

### Posso usare un dominio personalizzato?
Sì, puoi configurare un dominio personalizzato tramite le impostazioni del repository GitHub. Per dettagli, consulta la [documentazione di GitHub Pages](https://docs.github.com/en/pages).

### Il sito è sicuro?
Sì, GitHub Pages fornisce HTTPS per impostazione predefinita. Poiché il sito contiene contenuto statico, è intrinsecamente sicuro.

## Tecnico

### Quali tecnologie utilizza il sito?
- HTML5 per la struttura
- CSS3 per lo stile e il layout responsivo
- JavaScript vanilla per le animazioni e l'interattività
- GitHub Pages per l'hosting

### Perché non usa un framework moderno?
Per garantire velocità di caricamento, semplicità di manutenzione e nessuna dipendenza esterna. Un sito portfolio statico non ne ha bisogno.

### Il sito è responsivo?
Sì, il sito è completamente responsive e ottimizzato per desktop, tablet e dispositivi mobili.

### Come funzionano le animazioni?
Le animazioni sono implementate usando CSS3 e JavaScript vanilla. Puoi vederle durante lo scroll e le interazioni della pagina.

## Performance

### Come posso verificare le performance del sito?
Puoi usare strumenti come:
- Google PageSpeed Insights
- GTmetrix
- WebPageTest

Visita uno di questi siti e inserisci il link del sito.

### Il sito è lento?
Se il sito carica lentamente:
1. Svuota la cache del browser
2. Prova con un browser diverso
3. Controlla la velocità della tua connessione internet
4. Apri una issue se il problema persiste
